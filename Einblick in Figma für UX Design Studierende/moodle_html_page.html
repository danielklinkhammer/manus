Was ist Figma?

Figma ist ein cloudbasiertes Design-Tool, das speziell für die Zusammenarbeit entwickelt wurde. Es ermöglicht Designern, Entwicklern und anderen Stakeholdern, gemeinsam an UI/UX-Projekten zu arbeiten, unabhängig von ihrem Standort oder Betriebssystem.

Warum Figma für UX Design?

Figma hat sich in den letzten Jahren als eines der führenden Tools für UX/UI-Design etabliert, und das aus gutem Grund:

  • Kollaboration in Echtzeit: Mehrere Personen können gleichzeitig an einem Design arbeiten
  • Plattformübergreifend: Funktioniert im Browser auf jedem Betriebssystem
  • Komponenten und Styles: Erstellen Sie wiederverwendbare Designelemente für Konsistenz
  • Prototyping-Funktionen: Erstellen Sie interaktive Prototypen direkt in Figma
  • Umfangreiche Community: Zugang zu Tausenden von Vorlagen und Plugins
  • Kostenlose Version: Die Grundfunktionen sind kostenlos nutzbar

Überblick über die Benutzeroberfläche

Die Figma-Benutzeroberfläche besteht aus mehreren Hauptbereichen:

  • Werkzeugleiste: Enthält die wichtigsten Design-Werkzeuge (oben)
  • Ebenen-Panel: Zeigt die Hierarchie aller Designelemente (links)
  • Eigenschaften-Panel: Zeigt die Eigenschaften des ausgewählten Elements (rechts)
  • Canvas: Der Hauptarbeitsbereich, in dem Sie Designs erstellen (Mitte)
  • Seiten-Panel: Ermöglicht die Navigation zwischen verschiedenen Seiten oder Frames (links oben)

Hinweis: Figma bietet eine kostenlose Version, die für diesen Kurs ausreichend ist. Sie müssen lediglich ein Konto erstellen, um loszulegen.

Navigation in Figma

Die effiziente Navigation in Figma ist entscheidend für einen reibungslosen Workflow:

  • Zoomen: Mausrad oder Strg/Cmd + Mausrad
  • Verschieben: Leertaste + Mausziehen oder mittlere Maustaste
  • Auswählen: Klicken oder Auswahlrechteck ziehen
  • Mehrfachauswahl: Shift + Klicken auf Elemente
  • Alles auswählen: Strg/Cmd + A

Tipp: Drücken Sie die Taste "Z", um in den Zoom-Modus zu wechseln, und klicken Sie dann, um auf einen bestimmten Bereich zu zoomen.

Frames und Ebenen erstellen

Frames sind die Grundbausteine in Figma, vergleichbar mit Artboards in anderen Design-Tools:

  1. Frame erstellen: Wählen Sie das Frame-Tool (F) aus der Werkzeugleiste
  2. Gerätegröße auswählen: Wählen Sie eine vordefinierte Größe oder ziehen Sie einen benutzerdefinierten Frame
  3. Frames verschachteln: Ziehen Sie einen Frame in einen anderen, um eine Hierarchie zu erstellen
  4. Ebenen organisieren: Verwenden Sie das Ebenen-Panel, um Elemente zu gruppieren und zu ordnen

Grundlegende Formen und Textelemente

Figma bietet verschiedene Werkzeuge zum Erstellen von Grundformen und Text:

  • Rechteck-Tool (R): Erstellt Rechtecke und Quadrate
  • Ellipsen-Tool (O): Erstellt Kreise und Ellipsen
  • Linien-Tool (L): Erstellt gerade Linien
  • Polygon-Tool: Erstellt regelmäßige Polygone
  • Text-Tool (T): Fügt Textfelder hinzu

Um ein Element zu bearbeiten:

  1. Wählen Sie das Element aus
  2. Verwenden Sie das Eigenschaften-Panel rechts, um Farbe, Größe, Ausrichtung usw. zu ändern
  3. Verwenden Sie die Griffe am Element, um es zu skalieren oder zu drehen

Ausrichtung und Anordnung von Elementen

Figma bietet leistungsstarke Werkzeuge zur präzisen Ausrichtung von Elementen:

  • Intelligente Hilfslinien: Erscheinen automatisch beim Verschieben von Elementen
  • Ausrichtungsoptionen: Verfügbar in der oberen Werkzeugleiste, wenn mehrere Elemente ausgewählt sind
  • Abstände: Gleichmäßige Verteilung von Elementen horizontal oder vertikal
  • Raster und Lineale: Aktivierbar über das Menü Ansicht

Tipp: Halten Sie die Alt-Taste gedrückt, während Sie ein Element verschieben, um eine Kopie zu erstellen.

Styles und Komponenten: Grundkonzepte

Styles und Komponenten sind entscheidend für ein konsistentes Design:

  • Farb-Styles: Erstellen Sie wiederverwendbare Farben für Ihr Design-System
  • Text-Styles: Definieren Sie Schriftarten, Größen und Stile für verschiedene Texttypen
  • Effekt-Styles: Speichern Sie Schatten, Unschärfen und andere Effekte
  • Komponenten: Erstellen Sie wiederverwendbare UI-Elemente (z.B. Buttons, Karten)

Um einen Style zu erstellen:

  1. Wählen Sie ein Element aus
  2. Klicken Sie auf das Vier-Punkte-Symbol neben der Eigenschaft im Eigenschaften-Panel
  3. Wählen Sie "Style erstellen"
  4. Geben Sie einen Namen ein und klicken Sie auf "Erstellen"

Einführung in die Übung

In dieser praktischen Übung werden wir ein einfaches Wireframe für eine Wetter-App erstellen. Dieses Projekt ist ideal für Anfänger, da es grundlegende Figma-Funktionen abdeckt, ohne zu komplex zu sein.

Hinweis: Wireframes sind vereinfachte Darstellungen einer Benutzeroberfläche, die sich auf Struktur und Funktionalität konzentrieren, nicht auf visuelle Details.

Schritt 1: Frame erstellen

  1. Wählen Sie das Frame-Tool (F) aus der Werkzeugleiste
  2. Wählen Sie "iPhone 14 Pro / 15 Pro" aus den vordefinierten Größen (oder ein anderes Smartphone-Format)
  3. Klicken Sie auf den Canvas, um den Frame zu platzieren
  4. Benennen Sie den Frame "Wetter-App Hauptbildschirm" im Ebenen-Panel

Schritt 2: Hintergrund und Statusleiste hinzufügen

  1. Wählen Sie den Frame aus
  2. Ändern Sie die Hintergrundfarbe im Eigenschaften-Panel auf einen hellen Blauton (#E6F2FF)
  3. Erstellen Sie ein Rechteck (R) für die Statusleiste am oberen Rand des Frames:
    • Breite: Gesamte Framebreite
    • Höhe: 44px
    • Farbe: Weiß mit 80% Transparenz
  4. Fügen Sie ein Textelement (T) für die Uhrzeit hinzu (z.B. "9:41")
  5. Platzieren Sie es mittig in der Statusleiste

Schritt 3: Hauptinformationen hinzufügen

  1. Erstellen Sie ein Textelement (T) für den Stadtnamen:
    • Text: "Berlin"
    • Schriftgröße: 32px
    • Schriftart: System Default oder SF Pro
    • Ausrichtung: Zentriert
    • Position: Etwa 100px vom oberen Rand
  2. Fügen Sie ein Textelement für die Temperatur hinzu:
    • Text: "21°"
    • Schriftgröße: 72px
    • Schriftart: System Default oder SF Pro
    • Ausrichtung: Zentriert
    • Position: Unter dem Stadtnamen
  3. Fügen Sie ein Textelement für die Wetterbeschreibung hinzu:
    • Text: "Sonnig"
    • Schriftgröße: 18px
    • Schriftart: System Default oder SF Pro
    • Ausrichtung: Zentriert
    • Position: Unter der Temperatur

Schritt 4: Wetter-Icon hinzufügen

  1. Erstellen Sie einen Kreis mit dem Ellipsen-Tool (O):
    • Größe: 80x80px
    • Farbe: Gelb (#FFD700)
    • Position: Über der Temperatur
  2. Alternativ können Sie ein Icon aus der Figma Community importieren:
    • Klicken Sie auf das Menü "Ressourcen" (links oben)
    • Wählen Sie "Community"
    • Suchen Sie nach "Weather Icons"
    • Importieren Sie ein passendes Icon-Set
    • Ziehen Sie ein Sonnen-Icon auf Ihren Frame

Schritt 5: Tagesvorhersage-Bereich erstellen

  1. Erstellen Sie ein Rechteck (R) für den Vorhersage-Bereich:
    • Breite: Gesamte Framebreite
    • Höhe: 200px
    • Farbe: Weiß mit 90% Deckkraft
    • Eckenradius: 20px (im Eigenschaften-Panel)
    • Position: Im unteren Drittel des Frames
  2. Fügen Sie ein Textelement für die Überschrift hinzu:
    • Text: "7-Tage-Vorhersage"
    • Schriftgröße: 18px
    • Schriftart: System Default oder SF Pro
    • Schriftstärke: Medium oder Bold
    • Position: Oben im Vorhersage-Bereich mit 16px Abstand
  3. Erstellen Sie eine Tageszeile (für jeden Tag wiederholen):
    • Textelement für den Tag: "Mo" (16px)
    • Kleines Wetter-Icon (24x24px)
    • Textelement für die Temperatur: "19° / 24°" (16px)
    • Horizontale Linie als Trenner (1px Höhe, hellgrau)
  4. Duplizieren Sie die Tageszeile für weitere Tage (Di, Mi, Do, Fr)
  5. Verteilen Sie die Zeilen gleichmäßig im Vorhersage-Bereich

Schritt 6: Layout verfeinern

  1. Überprüfen Sie die Ausrichtung aller Elemente:
    • Wählen Sie alle Elemente im Hauptbereich aus und zentrieren Sie sie horizontal
    • Stellen Sie sicher, dass die Abstände zwischen den Elementen konsistent sind
  2. Fügen Sie Schatten oder Unschärfe-Effekte hinzu:
    • Wählen Sie den Vorhersage-Bereich aus
    • Fügen Sie im Eigenschaften-Panel unter "Effekte" einen leichten Schatten hinzu
  3. Überprüfen Sie die Lesbarkeit:
    • Stellen Sie sicher, dass alle Texte gut lesbar sind
    • Passen Sie bei Bedarf Farben und Kontraste an

Tipp: Verwenden Sie die Tastenkombination Strg/Cmd + D, um Elemente schnell zu duplizieren, nachdem Sie sie ausgewählt haben.

Einfache Interaktionen hinzufügen

Figma ermöglicht es Ihnen, interaktive Prototypen zu erstellen, ohne den Editor zu verlassen:

  1. Erstellen Sie einen zweiten Frame für einen Detailbildschirm:
    • Duplizieren Sie den Hauptbildschirm (Strg/Cmd + D)
    • Ändern Sie einige Elemente, um ihn zu unterscheiden
    • Platzieren Sie ihn rechts neben dem ersten Frame
  2. Wechseln Sie in den Prototyping-Modus:
    • Klicken Sie auf den Tab "Prototyp" im rechten Panel
  3. Erstellen Sie eine Verbindung:
    • Wählen Sie ein Element im ersten Frame aus (z.B. eine Tageszeile)
    • Ziehen Sie den Pfeil-Connector (kleiner blauer Kreis) zum zweiten Frame
    • Wählen Sie die Interaktionseinstellungen:
      • Interaktion: Bei Klick
      • Animation: Smart Animate
      • Dauer: 300ms
  4. Erstellen Sie eine Rückverbindung:
    • Wählen Sie ein Element im zweiten Frame aus (z.B. einen "Zurück"-Button)
    • Verbinden Sie es mit dem ersten Frame

Prototyp-Vorschau und Teilen

So können Sie Ihren Prototyp testen und teilen:

  1. Starten Sie die Vorschau:
    • Klicken Sie auf den "Abspielen"-Button in der oberen rechten Ecke
    • Oder drücken Sie Strg/Cmd + Alt + Enter
  2. Testen Sie die Interaktionen:
    • Klicken Sie auf die interaktiven Elemente
    • Überprüfen Sie, ob die Übergänge wie erwartet funktionieren
  3. Teilen Sie den Prototyp:
    • Klicken Sie auf die Schaltfläche "Teilen" in der oberen rechten Ecke
    • Wählen Sie die Freigabeeinstellungen (Ansicht oder Bearbeitung)
    • Kopieren Sie den Link oder laden Sie bestimmte Personen ein

Hinweis: Mit der kostenlosen Version von Figma können Sie unbegrenzt viele Projekte erstellen und mit bis zu zwei Bearbeitern zusammenarbeiten.

Zusammenarbeit in Echtzeit demonstrieren

Figma zeichnet sich durch seine Kollaborationsfunktionen aus:

  • Mehrere Bearbeiter: Mehrere Personen können gleichzeitig am selben Dokument arbeiten
  • Cursor-Verfolgung: Sehen Sie, wo andere Benutzer arbeiten
  • Bearbeitungsrechte: Steuern Sie, wer was bearbeiten kann

So laden Sie Kollaborateure ein:

  1. Klicken Sie auf die Schaltfläche "Teilen" in der oberen rechten Ecke
  2. Geben Sie die E-Mail-Adressen der Personen ein, mit denen Sie zusammenarbeiten möchten
  3. Wählen Sie die entsprechenden Berechtigungen (Ansicht, Kommentieren, Bearbeiten)
  4. Klicken Sie auf "Einladen"

Kommentare und Feedback

Figma bietet leistungsstarke Feedback-Tools:

  1. Aktivieren Sie den Kommentar-Modus:
    • Klicken Sie auf das Sprechblasen-Symbol in der oberen Werkzeugleiste
    • Oder drücken Sie C
  2. Hinterlassen Sie einen Kommentar:
    • Klicken Sie auf den Bereich, den Sie kommentieren möchten
    • Geben Sie Ihren Kommentar ein
    • Drücken Sie Enter, um den Kommentar zu speichern
  3. Antworten und Auflösen:
    • Klicken Sie auf einen vorhandenen Kommentar, um zu antworten
    • Klicken Sie auf das Häkchen, um einen Kommentar als gelöst zu markieren

Tipp: Verwenden Sie @Erwähnungen in Kommentaren, um bestimmte Teammitglieder zu benachrichtigen.

Praktische Übungsprojekte

Um Ihre Figma-Fähigkeiten zu verbessern, empfehlen wir folgende Übungsprojekte:

  1. Erweitern Sie die Wetter-App:
    • Fügen Sie weitere Screens hinzu (Einstellungen, Standortauswahl, etc.)
    • Erstellen Sie ein vollständiges Navigationskonzept
    • Experimentieren Sie mit verschiedenen Farbschemata
  2. Erstellen Sie ein einfaches Design-System:
    • Definieren Sie Farb- und Typografie-Styles
    • Erstellen Sie wiederverwendbare Komponenten (Buttons, Karten, etc.)
    • Dokumentieren Sie Ihre Komponenten
  3. Redesign einer bestehenden App:
    • Wählen Sie eine App, die Sie häufig nutzen
    • Analysieren Sie die Benutzeroberfläche
    • Erstellen Sie ein verbessertes Design in Figma

Tipp: Regelmäßiges Üben ist der Schlüssel zur Beherrschung von Figma. Versuchen Sie, täglich mindestens 30 Minuten mit dem Tool zu arbeiten.
