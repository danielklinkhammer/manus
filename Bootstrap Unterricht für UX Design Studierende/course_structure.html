Kursübersicht

Dieser Bootstrap-Kurs ist speziell für UX Design Studierende mit wenig technischem Vorwissen konzipiert. In zwei 3-stündigen Vorlesungen werden die Grundlagen von Bootstrap vermittelt und der Übergang von Figma Dev Mode zu Bootstrap sichergestellt. Als Entwicklungsumgebung wird Visual Studio Code verwendet.

Kursziele

  • Verständnis der Bootstrap-Grundlagen und -Komponenten
  • Fähigkeit, Figma-Designs in Bootstrap-Code umzusetzen
  • Einrichtung und Verwendung von Visual Studio Code für Bootstrap-Entwicklung
  • Erstellung responsiver Webseiten mit Bootstrap
  • Anwendung von Bootstrap in realen Projekten

Hinweis

Alle Studierenden sollten vor Kursbeginn Visual Studio Code installieren. Eine detaillierte Anleitung finden Sie unter Visual Studio Einrichtung.
