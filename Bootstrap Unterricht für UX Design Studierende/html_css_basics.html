Einführung

Bevor wir mit Bootstrap beginnen, ist es wichtig, die Grundlagen von HTML und CSS zu verstehen. Diese Wiederholung soll Ihnen helfen, die wichtigsten Konzepte aufzufrischen, damit Sie Bootstrap effektiv nutzen können.

Hinweis

Diese Wiederholung ist für Studierende mit wenig technischem Vorwissen gedacht. Wenn Sie bereits Erfahrung mit HTML und CSS haben, können Sie direkt zur Bootstrap-Einführung übergehen.

HTML Grundlagen

HTML (Hypertext Markup Language) ist die Standardsprache für die Erstellung von Webseiten. HTML beschreibt die Struktur einer Webseite semantisch und besteht aus einer Reihe von Elementen, die dem Browser mitteilen, wie der Inhalt dargestellt werden soll.

HTML-Dokument-Struktur

Ein grundlegendes HTML-Dokument hat folgende Struktur:

<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meine Webseite</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <!-- Hier kommt der Inhalt der Webseite -->
    <h1>Willkommen auf meiner Webseite</h1>
    <p>Dies ist ein Absatz.</p>
</body>
</html>

Wichtige HTML-Elemente

Element Beschreibung Beispiel
<h1> bis <h6> Überschriften verschiedener Ebenen <h1>Hauptüberschrift</h1>
<p> Absatz <p>Dies ist ein Absatz.</p>
<a> Hyperlink <a href="https://example.com">Link</a>
<img> Bild <img src="bild.jpg" alt="Beschreibung">
<div> Container für andere Elemente <div>Inhalt</div>
<span> Inline-Container für Text <span>Text</span>
<ul>, <ol>, <li> Listen (ungeordnet, geordnet, Listenelement) <ul><li>Element</li></ul>
<form>, <input>, <button> Formularelemente <form><input type="text"></form>

Semantische HTML-Elemente

Semantische Elemente beschreiben klar, was ihr Inhalt bedeutet. Sie verbessern die Zugänglichkeit und SEO Ihrer Webseite.

<header>

Definiert den Kopfbereich einer Webseite oder eines Abschnitts.

<nav>

Definiert einen Navigationsbereich.

<main>

Definiert den Hauptinhalt einer Webseite.

<section>

Definiert einen thematischen Abschnitt.

<article>

Definiert einen eigenständigen Inhalt.

<footer>

Definiert den Fußbereich einer Webseite oder eines Abschnitts.

CSS Grundlagen

CSS (Cascading Style Sheets) wird verwendet, um das Erscheinungsbild von HTML-Elementen zu gestalten. Mit CSS können Sie Farben, Schriftarten, Abstände, Layouts und vieles mehr definieren.

CSS-Syntax

CSS besteht aus Selektoren und Deklarationen:

selektor {
    eigenschaft: wert;
    eigenschaft: wert;
}

CSS-Selektoren

Selektor Beschreibung Beispiel
Element Wählt alle Elemente eines bestimmten Typs aus p { color: blue; }
Klasse Wählt Elemente mit einer bestimmten Klasse aus .button { background: red; }
