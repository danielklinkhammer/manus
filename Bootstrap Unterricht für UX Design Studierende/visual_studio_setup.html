Visual Studio Einrichtung

Eine Schritt-für-Schritt-Anleitung zur Einrichtung von Visual Studio Code für Bootstrap-Entwicklung.

Inhaltsverzeichnis
Einführung Installation Erweiterungen Konfiguration Bootstrap-Einrichtung Workflow Tastenkürzel Fehlerbehebung

Einführung

Visual Studio Code (VS Code) ist ein leichtgewichtiger, aber leistungsstarker Quellcode-Editor, der auf Ihrem Desktop läuft und für Windows, macOS und Linux verfügbar ist. Er bietet integrierte Unterstützung für JavaScript, TypeScript und Node.js und verfügt über ein reichhaltiges Ökosystem von Erweiterungen für andere Sprachen und Runtimes.

Hinweis

In diesem Kurs verwenden wir Visual Studio Code als Entwicklungsumgebung für unsere Bootstrap-Projekte. Es ist kostenlos und bietet hervorragende Unterstützung für HTML, CSS und JavaScript.

Warum Visual Studio Code?

VS Code bietet mehrere Vorteile für die Bootstrap-Entwicklung:

Leichtgewichtig und schnell

VS Code startet schnell und reagiert zügig, auch bei größeren Projekten.

Erweiterbar

Ein reichhaltiges Ökosystem von Erweiterungen ermöglicht die Anpassung an Ihre Bedürfnisse.

Hervorragende Unterstützung für Web-Technologien

Integrierte Unterstützung für HTML, CSS und JavaScript mit IntelliSense und Debugging.

Integriertes Terminal

Ein integriertes Terminal ermöglicht die Ausführung von Befehlen ohne Wechsel der Anwendung.

Installation

Die Installation von Visual Studio Code ist einfach und unkompliziert. Folgen Sie diesen Schritten:

  1. Besuchen Sie die offizielle VS Code-Website.
  2. Klicken Sie auf die Schaltfläche "Download for Windows".
  3. Führen Sie die heruntergeladene Installationsdatei aus.
  4. Folgen Sie den Anweisungen des Installationsassistenten. Die Standardeinstellungen sind in der Regel ausreichend.
  5. Starten Sie VS Code nach Abschluss der Installation.

Tipp: Aktivieren Sie während der Installation die Option "Add to PATH", um VS Code von der Kommandozeile aus starten zu können.

  1. Besuchen Sie die offizielle VS Code-Website.
  2. Klicken Sie auf die Schaltfläche "Download for Mac".
  3. Öffnen Sie die heruntergeladene ZIP-Datei.
  4. Ziehen Sie die VS Code-Anwendung in den Anwendungsordner.
  5. Starten Sie VS Code aus dem Anwendungsordner oder über Spotlight (Cmd+Space).

Tipp: Um VS Code von der Kommandozeile aus zu starten, öffnen Sie VS Code und drücken Sie Cmd+Shift+P, geben Sie "shell command" ein und wählen Sie "Shell Command: Install 'code' command in PATH".

Die Installation unter Linux variiert je nach Distribution. Hier sind die gängigsten Methoden:

Debian/Ubuntu:
sudo apt update
sudo apt install software-properties-common apt-transport-https wget
wget -q https://packages.microsoft.com/keys/microsoft.asc -O- | sudo apt-key add -
sudo add-apt-repository "deb [arch=amd64] https://packages.microsoft.com/repos/vscode stable main"
sudo apt update
sudo apt install code
Fedora/RHEL:
sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/yum.repos.d/vscode.repo'
sudo dnf check-update
sudo dnf install code
Arch Linux:
sudo pacman -S code

Tipp: Alternativ können Sie auch die .deb- oder .rpm-Pakete von der offiziellen Website herunterladen und manuell installieren.
