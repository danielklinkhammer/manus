Praktische Übungen

Hands-on Übungen und Projekte zum Erlernen von Bootstrap für UX Design Studierende.

Inhaltsverzeichnis

Einführung

Praktische Übungen sind ein wesentlicher Bestandteil des Lernprozesses. In diesem Abschnitt finden Sie eine Reihe von Übungen, die Ihnen helfen, die Konzepte von Bootstrap zu verstehen und anzuwenden. Die Übungen sind nach Schwierigkeitsgrad geordnet und bauen aufeinander auf.

Hinweis

Für alle Übungen stehen Starter-Code und vollständige Lösungen zur Verfügung. Es wird empfohlen, zuerst zu versuchen, die Übungen selbstständig zu lösen, bevor Sie die Lösungen ansehen.

Voraussetzungen

Bevor Sie mit den Übungen beginnen, stellen Sie sicher, dass Sie:

  • Die HTML/CSS Grundlagen verstanden haben
  • Die Bootstrap-Einführung durchgearbeitet haben
  • Visual Studio Code gemäß der Einrichtungsanleitung installiert haben

Wie man die Übungen bearbeitet

  1. Laden Sie den Starter-Code für die jeweilige Übung herunter
  2. Öffnen Sie den Code in Visual Studio Code
  3. Lesen Sie die Anweisungen sorgfältig durch
  4. Bearbeiten Sie die Aufgaben Schritt für Schritt
  5. Testen Sie Ihre Lösung in verschiedenen Browsern und auf verschiedenen Geräten
  6. Vergleichen Sie Ihre Lösung mit der bereitgestellten Musterlösung

Übung 1: HTML/CSS Grundlagen

Diese Übung dient dazu, Ihre HTML- und CSS-Kenntnisse aufzufrischen, bevor wir mit Bootstrap beginnen.

Aufgabe: Erstellen einer einfachen Webseite
Ziel:

Erstellen Sie eine einfache Webseite mit HTML und CSS, die folgende Elemente enthält:

  • Eine Überschrift (h1)
  • Einen Absatz mit Text
  • Ein Bild
  • Eine ungeordnete Liste mit drei Elementen
  • Einen Link zu einer externen Webseite
Anforderungen:
  • Verwenden Sie semantisches HTML
  • Stylen Sie die Webseite mit CSS (Farben, Schriftarten, Abstände)
  • Stellen Sie sicher, dass die Webseite gut aussieht und lesbar ist
Starter-Code:
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Übung 1: HTML/CSS Grundlagen</title>
    <style>
        /* Hier kommen Ihre CSS-Styles */
    </style>
</head>
<body>
    <!-- Hier kommt Ihr HTML-Code -->
</body>
</html>

Tipp

Achten Sie auf die korrekte Struktur Ihres HTML-Dokuments und vergessen Sie nicht, das Viewport-Meta-Tag einzufügen, um die Responsivität zu gewährleisten.

Übung 2: Bootstrap Grid

In dieser Übung lernen Sie, das Bootstrap-Grid-System zu verwenden, um responsive Layouts zu erstellen.

Aufgabe: Erstellen eines responsiven Layouts
Ziel:

Erstellen Sie ein responsives Layout mit dem Bootstrap-Grid-System, das folgende Anforderungen erfüllt:

  • Eine Kopfzeile, die die gesamte Breite einnimmt
  • Einen Hauptbereich mit drei Spalten auf Desktop-Geräten
  • Auf Tablets sollten die drei Spalten zu zwei Spalten werden (2+1)
  • Auf Mobilgeräten sollten alle Spalten untereinander angezeigt werden
  • Eine Fußzeile, die die gesamte Breite einnimmt
Anforderungen:
  • Verwenden Sie Bootstrap 5 über CDN
  • Verwenden Sie die Bootstrap-Grid-Klassen (.container, .row, .col-*)
  • Fügen Sie in jede Spalte etwas Beispielinhalt ein
  • Testen Sie das Layout auf verschiedenen Bildschirmgrößen
Starter-Code:
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Übung 2: Bootstrap Grid</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet">
    <style>
        /* Zusätzliche Styles für die Visualisierung */
        .header, .footer {
            background-color: #f8f9fa;
            padding: 20px;
            margin-bottom: 20px;
        }
        .footer {
            margin-top: 20px;
            margin-bottom: 0;
        }
        .content {
            background-color: #e9ecef;
            padding: 20px;
            margin-bottom: 20px;
            min-height: 200px;
        }
    </style>
</head>
<body>
    <!-- Hier kommt Ihr Bootstrap-Grid-Code -->
    
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Tipp

Denken Sie daran, dass das Bootstrap-Grid-System auf 12 Spalten basiert. Verwenden Sie die Breakpoint-Präfixe (sm, md, lg, xl), um das Verhalten auf verschiedenen Bildschirmgrößen zu steuern.
